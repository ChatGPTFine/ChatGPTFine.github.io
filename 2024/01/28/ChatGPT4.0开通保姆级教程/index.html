ChatGPT4.0开通保姆级教程

本文分享一下本人(以及若干ChatGPT Plus爱好者+群友)亲测有效的ChatGPT Plus支付宝付费版升级流程，以及体验ChatGPT4.0的感受，只能说贵但是值得。

实操步骤

大前提： 只要能科学上网即可~

  • 注册虚拟信用卡
  • 一键注册 OpenAI 账户
  • 一键升级 ChatGPT Plus

注册WildCard虚拟信用卡

注册账号：WildCard | 一分钟轻松开卡，轻松订阅海外服务 点击链接注册账号

付款方式：支付宝，微信

提现方式：随时，自动返还到支付宝账号

重点：这个WildCard虚拟信用卡，不仅可以用来订阅 ChatGPT，一些常见的国外付费订阅软件都可以使用（例如：OPenAI API，Midjourney，Apple Store，Google Play等等），强烈推荐开一个~

注册 OpenAI 账户

注：如果已经有 OpenAI 账户可跳至下一步

  1. 首先点击一键注册 OpenAI 账户

  2. 购买OpenAI账户(约3.53 元)

  3. 注册OpenAI账户

  4. 注册完成后要及时登录,设置一些信息

注：注册完成后要及时登录（十分钟内）,设置个人信息

此时你就拥有一个 OpenAI 账户啦，可登录官网：ChatGPT 官网

顺利的使用chatGPT3.5吧！

升级ChatGPT Plus流程

想要体验更强的ChatGPT Plus的小伙伴接着往下走

  1. 点击一键升级ChatGPT

  2. 在WildCard内进行充值，开通Plus版本一个月需要20美金

    这里需要设置支付密码，支付宝直接付款即可，如果支付完后悔还能提现，但是需要一点点手续费~

  3. 回到升级页面点击Upgrade to Plus：

    不需要填写支付信息，直接将链接复制到WildCard开卡平台请求升级即可，超级方便！！

  4. 尽情享受ChatGPT4.0的强大功能吧！！
